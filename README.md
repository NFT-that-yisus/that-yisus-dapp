# Cómo utilizar este proyecto.

Antes de proceder, situate en tu escritorio. Puedes hacerlo con
```bash
cd desktop
```

1. Clona este repositorio
```bash
git clone https://github.com/NFT-that-yisus/that-yisus-dapp.git
```

2. Entra a la carpeta del proyecto
```bash
cd that-yisus-dapp
```
3. Instala los paquetes necesarios
```bash
npm install
```

4. Ejecuta el proyecto
```bash
npm run dev
```

Recomendamos a la hora de utilizar este código para tu proyecto, saber que es lo que haces.

Estos son los archivos que debes modificar.

1. /public/config/abi.json
Una vez verificado tu Smart Contract copia el abi y ponlo en el archivo.

2. EN PROCESO
